云计算巷战在即 微软押宝开源

2014-03-19

2014年是云计算的分水岭。

此前，国内外IT、互联网各大公司的云计算都在发力，所属领域各所建树，但主要成果大多在私有云中体现。2014年之后，几大国际巨头云计算公有云平台已在华落地，正式开启了巷战模式。

大幕即将拉开，微软确定了以“开源”作为自己云计算的主攻方向。挥泪转型云计算的 IBM试图在新领域自救，已然世界云计算领域老大的亚马逊也按部就班开启了在华工作。

 巷战来临

云计算概念已经热炒多年，以微软、IBM为代表的IT巨头和以谷歌亚马逊为代表互联网国际巨头，国内阿里、盛大、华为等诸多公司也都在纷纷布局云计算，各大公司在私有云拼杀多年，各有千秋，而公有云平台在国内的巷战时代也即将来临。

2013年，微软、IBM、亚马逊八仙过海、各显神通，通过与本地不同的合作伙伴合作的方式实现了云计算平台的落地。

去年5月，微软宣布联手上海市政府、世纪互联把其公有云服务带入中国，正式实现了其Windows Azure平台的落地，成为第一个落地成功的国际公有云平台。

7月，IBM宣布与首都在线签署公有云长期战略合作协议，双方在公有云领域展开全方位战略合作，共同构建公有云平台，IBM承接该平台的设计、研发、构建，并且全面参与该平台建成后的管理工作。

12月底，亚马逊中国在北京召开发布会，正式宣布公有云服务 AWS(Amazon Web Service)通过与宽带资本旗下云基地的战略合作，实现 AWS 在中国的落地。中国将是亚马逊 AWS 在亚太地区的第 4 个区域，也是全球范围内的第 10 个区域。

巨头来临，国内百度云计算、腾讯的微云、360云纷纷跟着热闹了一阵，掀起1元2T存储空间，免费无限量存储空间等针对个人用户的服务。不过，个人用户凑凑热闹没问题。但企业级用户的决策就不这么容易了。

时至今日，因为暂时没有统一的认识和标准，连各大CTO们都无法确切说明，什么样的云服务才是好的服务。

近期，微软、IBM、亚马逊都在争相公布本地案例积累，微软还搭上2014年春晚的网络直播。但如何把战果扩大，尽快争取得到更多本地客户的认可，巨头们需要使出各自的杀手锏。

微软押宝开源

微软此前对“开源”的态度摇摆不明，但对Windows Azure“开源”的措施态度明朗，微软专门成立了名字为“开放”的子公司，致力于在微软和非微软技术之间搭建桥梁。

开源还是不开源只是一种游戏规则，并不能决定成功还是失败。但显然，Windows Azure平台将有助于打造更健康的业内生态圈，更适合时下需求。

据了解，WindowsAzure对开源的支持主要体现在两个方面：第一，在Windows Azure上可以部署和运行虚拟机，支持开源操作系统，也支持在Windows和开源操作系统上运行各类开源软件以及开源数据库等。第二，WindowsAzure对开源的支持体现在对于开源开发的支持，目前支持流行的开源编程语言如Java、Node.js、Ruby、Python。Windows Azure SDK自身也是开源的，而且Windows Azure内部也用到了一些开源产品。

同时，在WindowsAzure上除了能够运行开源操作系统外，还可以运行其他的开源软件，如MongoDB这样的开源数据库。此外，微软和开源社区有一定的合作，在开源社区里提供了完整的文档，如告诉大家怎样在IaaS和PaaS的虚拟机上安装MongoDB。开发者可以在Github上下载源代码，其中包括一些命令行工具。

此外，微软windowsazure移动服务支持安卓平台，可更轻松地将客户端和移动应用程序连接至可扩展的云后端。2013年4月，微软宣布向windows azure mobile services添加对html5应用程序的支持，让企业有更多选项，轻松创建可在云中、在各种设备上快速扩展的应用程序。

微软开放技术有限公司总裁蒋博励公开表示，Windows Azure本质上是互联网服务，主要通过浏览器向用户提供服务和接口，这从根本上决定了Windows Azure的开源“DNA”。因为只有提供最大的开放性才能够获得最广泛的用户。

 各执利器

目前看来，亚马逊AWS是云计算领域的霸主。据Gartner 的数据显示，AWS的市场占有率是其他 14 家主要公司总和(谷歌除外)的五倍。

尽管亚马逊从未公布具体业绩，但据 Macquarie 分析师的预计，去年亚马逊 AWS 部门营收接近 40 亿美元。尽管这与电商部门超过 600 亿美元的销售额尚且无法相提并论，但亚马逊 CEO 杰夫·贝佐斯却预计，未来AWS 将成为亚马逊最大的业务。美国投资银行 Evercore Partners 的分析师 Ken Sena 预计，到 2015 年时，AWS 的估值或达到 500 亿美元。与此同时，微软Windows Azure平台目前收入在10亿美元左右。

但是，鉴于跨国公司借中国子公司来落地运营某个复杂产品通常会遭遇“水土不服”，而AWS落地之后如何构建生态圈、如何开拓本地市场，并未对外披露过多信息。摆在 AWS 面前最大的问题在于本土化，从产品研发、技术支持到销售支持都是未来业务的关键。

与亚马逊的春风得意相比，IBM的云计算之路伴随着血泪，IBM在宣布投资12亿美元扩展其全球云计算服务的同一个月，也宣布未来的财报还会计入10亿美元的“劳动力重置”(就是裁员)支出。

“蓝色巨人”下定决心自我开刀，迎接转变了。同亚马逊相比，IBM在华多年的客户积累和渠道培养，是IBM目前最大的凭仗。

IBM多年来，一直服务于政府、大型企业客户，IBM已经在智慧城市、电信、零售、金融等行业帮助众多客户实现了高价值的云计算。为此，IBM也有不错的回报，2013年IBM单季云计算收入达到10亿美元。据统计，IBM拥有超过5000个私有云应用，2012年比2011年同比增长超过100%。私有云的技术堆栈能否为IBM的公有云铺平道路，还需要时间检验。

中国是个神奇的赛场。除了产品、销售、服务之外，还可能有其他“外挂”来改变游戏规则。所以，笔者不妄言结局。大幕拉开，大家备好沙发爆米花，等着看，走着瞧。
